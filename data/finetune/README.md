# Fine-Tuning Benchmark

22,500 train + 1,000 test examples generated by src/dataset/generate_dataset.py.

## Generate
```bash
python src/dataset/generate_dataset.py --n 25000 --seed 42 \
    --out data/finetune/cot/regex_dfa_dataset.jsonl
```

## Tiers
| Tier | Share | DFA States | Description                        |
|------|-------|------------|------------------------------------|
| 1    | ~15%  | 2          | Direct symbol mapping              |
| 2    | ~25%  | 2–4        | Kleene star, plus, optional        |
| 3    | ~35%  | 3–6        | Alternation and branching          |
| 4    | ~25%  | 4–18       | Full NFA-to-DFA subset construction|

## Formats
- cot/   : full Thompson→subset→Hopcroft trace + final DFA (Appendix C.2)
- nocot/ : final minimised DFA transition table only
